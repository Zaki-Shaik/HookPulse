# HookPulse - A Asynchronous, Durable Webhook Platform

## Why Does this exit
This is a purely a learning project, because I got curious with how Stripe manages
their payments. Wanted to build this so I can explore the various challenges 
one can run into a system that needs to reliably deliver events to other services.

Having said that, this project when completed can still be used as a intermediary for any 
services that want to manage their webhook events in one place.
